ZooBuilder Gamey UI Package
===========================

Files:
- index.html
- styles.css
- script.js

How to use:
1. Upload the `frontend` folder contents (these files) to your hosting or replace the current frontend files in your repo.
2. Open `index.html` in a browser or inside Telegram (for full WebApp features).
3. The UI is a demo stub: clicking enclosures starts prompt-based interactions; integrate with backend as needed.

To update your GitHub repo locally:
git add frontend/*
git commit -m "Update gamey UI"
git push
